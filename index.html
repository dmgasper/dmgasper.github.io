What is My Story?

 In my philosophy, engineering and invention is synonymous with art, in that all involve the creative process. I seek solutions that push the limits of typicality and explore new possibilities. In both my work and personal life, I work to integrate this creative process while keeping the result of my work effective and communicable.

 My career objective is to channel my passion for creative problem solving into software development. I aim to use two years of professional and semi-professional application development experience, a hackathon-style team programming challenge, and 1.5 years of internship experience to enter the software industry and contribute out of the box thinking to your engineering team.

 Since my first development project, an implementation of the game Foosball written in Java when I was twelve years old, I have developed or contributed to several applications on a professional and non-professional basis.

What are My Projects to Date?

Gimme Foosball
2012

I created this game when I was 12 years old to emulate the tabletop soccer game Foosball. The game used Java for all functionality including 2D graphics, sound, and a computerized opponent. The website used PHP and HTML and supported user logins and a Paypal integration so that users could purchase the "premium" version of the game.

Williams Pace Calculator
2019-

My running coach needed a mobile app that could be used by coaches and runners to calculate their split times for various running distances based on their race performance. I was hired to develop this app for Android from the ground up and publish the app on the Google Play store using his design and specifications. The app is built using Java for Android and XML for the user interface layout. The app is sold internationally in the United States and Japan. Since the app's creation, I have continued to provide maintenance and support.

Project website
Mynd Social
2021-2022

Mynd Social is a mobile app prototype designed to help young people establish balance with and find meaningful identities around social media platforms. I worked with a grant-winning team on this project in a hackathon-style development challenge. I built the app prototype using React Native and AWS and I built the app's landing page using React.

Know Before You Go
2021-2022

I worked with a team of eight OSU students to build a full-stack application to inform Corvallis, OR area trail users of trailhead capacity. With this project my team strove to improve community access to nature and reduce overcrowding of fragile nature areas. I developed the middleware for this project using AWS Lambda functions to create API endpoints for updating and accessing trailhead data as well as a NoSQL database for storing this data.

Scrollnt
2023

During my undergraduate program, I performed a research study on engagement with social media platforms. In this study, participants (all OSU students) were asked to complete a study task while being shown a simulated social media feed and simulated popup notifications. To assist in data collection, I developed a React frontend and Node.JS backend to display the study materials and social media feed. In addition, this application recorded mouse movement data and stored the data in an Azure Cosmos DB for scientific analysis.
