About Me

 To me, engineering and invention is synonymous with art, in that all involve the creative process. As such, I seek solutions for any challenge that I face, be it an engineering project at work or a situation in my personal life, that are not only effective and communicable but also that may push the limits of typicality and that explore new possibilities.

 My career objective is to channel my passion for creative problem solving and years of freelance app development experience, hackathon-style team programming challenges, and internship experience to enter the software industry and contribute out of the box thinking to your engineering team.

 Since my first development project, an implementation of the game Foosball written in Java when I was twelve years old, I have developed or contributed to several applications on a professional and non-professional basis.

My Projects

Gimme Foosball
2012

I created this game when I was 12 years old to emulate the tabletop soccer game Foosball. The game used Java for all functionality including 2D graphics, sound, and a computerized opponent. The website used PHP and HTML and supported user logins and a Paypal integration so that users could purchase the "premium" version of the game.

Williams Pace Calculator
2019

My running coach needed a mobile app that could be used by coaches and runners to calculate their split times for various running distances based on their race performance. I was hired to develop this app for Android from the ground up and publish the app on the Google Play store using his design and specifications. The app was built using Java for Android and XML for the user interface layout.

Know Before You Go
2021-2022

I worked with a team of eight OSU students to build a full-stack application to inform Corvallis, OR area trail users of trailhead capacity. With this project my team strove to improve community access to nature and reduce overcrowding of fragile nature areas. I developed the middleware for this project using AWS Lambda functions to create API endpoints for updating and accessing trailhead data as well as a NoSQL database for storing this data.

Scrollnt
2023

During my undergraduate program, I performed a research study on engagement with social media platforms. In this study, participants (all OSU students) were asked to complete a study task while being shown a simulated social media feed and simulated popup notifications. To assist in data collection, I developed a React frontend and Node.JS backend to display the study materials and social media feed. In addition, this application recorded mouse movement data and stored the data in an Azure Cosmos DB for scientific analysis.
